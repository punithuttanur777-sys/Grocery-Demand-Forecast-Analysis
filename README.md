# Grocery Demand Forecast Analysis

## Project Overview
This project focuses on forecasting future grocery demand using historical sales data. The Facebook Prophet model is used to identify trends, seasonality, and future sales patterns, helping businesses make data-driven inventory decisions.

## Problem Statement
Retail businesses often face challenges such as stock shortages, overstocking, and inventory wastage due to inaccurate demand forecasting.

## Solution
This project analyzes historical grocery sales data and predicts future demand using the Prophet forecasting model. The results are visualized through an interactive Tableau dashboard.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Facebook Prophet
- Tableau
- VS Code

## Dataset
Corporación Favorita Grocery Sales Dataset (Ecuador)

## Key Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Demand Forecasting
- Tableau Dashboard Visualization
- Business Insights Generation

## Results
- Forecasted future grocery demand
- Identified sales trends and seasonality
- Improved inventory planning insights
